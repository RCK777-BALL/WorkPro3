# Role Permissions

## Core Permissions
- `assets.read`, `assets.write`, `assets.delete`
- `workorders.read`, `workorders.write`, `workorders.approve`
- `pm.read`, `pm.write`, `pm.delete`
- `inventory.read`, `inventory.manage`, `inventory.purchase`
- `roles.read`, `roles.manage`

## Notes
- Tenant scope is enforced via `tenantId`.
- Site scope is enforced where configured.
